# Osi Server - Client

## Создайте сервер с использованием ServerSocket.
При входящем соединении на сервер, примите его, прочитайте из него строку. Выведите её на экран вместе с номером порта, с которого это соединение пришло.
Вместе с этим в другом методе (это может быть другой класс со вторым main методом или отдельный поток) подключитесь к этому серверу и отправьте ему одну строку.

## Напишите игру в города.

Игроки по очереди называют города, главное правило - следующий город должен начинаться на ту же букву, на которую заканчивался последний названный город.
Каждый игрок подключается один раз и вводит только один город. Мы доверяем, что слово, введённое игроком - название города. Можно считать, что все города вводятся 
с маленькой буквы.
